## 🇬🇧 English

### 📌 Overview

This project presents an interactive and responsive **editorial workflow diagram** that illustrates the complete lifecycle of a manuscript — from submission to publication.

It is designed for:
- ✍️ Authors  
- 🧪 Reviewers  
- 🧑‍💼 Editors  
- 📚 Readers and researchers  

The workflow is aligned with international academic publishing standards and implemented in **OJS (Open Journal Systems)** environments.

---

### 🧩 Workflow Stages

1. **Submission and Initial Screening**  
2. **Peer Review Process**  
3. **Copyediting and Style Correction**  
4. **Production and Publication**  

---

### 🔐 Peer Review Policy

This workflow follows a **double-blind peer review model**:

> Neither authors nor reviewers know each other's identities during the evaluation process.

This ensures impartiality, fairness, and academic integrity.

---

### ⚙️ Features

- Flowchart with standard symbols (start/end, process, decision)  
- Clear **YES / NO decision paths**  
- Role-based highlighting (Author, Editor, Reviewer, etc.)  
- Timeline indicators (KPIs)  
- Responsive design (mobile-friendly)  
- Built with **TailwindCSS**

## 🇪🇸 Español

### 📌 Descripción

Este repositorio contiene la **visualización del flujo editorial** utilizada en las revistas gestionadas por [Chaski Group](https://chaskigroup.org/).

El objetivo es representar de forma clara todo el proceso editorial, desde el envío del manuscrito hasta su publicación final.

Está dirigido a:
- ✍️ Autores  
- 🧪 Revisores  
- 🧑‍💼 Editores  
- 📚 Lectores e investigadores  

---

### 🧩 Etapas del flujo

1. **Envío y evaluación inicial**  
2. **Revisión por pares**  
3. **Corrección de estilo**  
4. **Producción y publicación**

### 🔐 Política de revisión

Este flujo utiliza un modelo de **revisión por pares doble ciego**:

> Ni los autores ni los revisores conocen sus identidades durante el proceso de evaluación.
